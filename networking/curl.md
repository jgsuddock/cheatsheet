# Curl

## Request Types

GET Request

```bash
curl https://example.com
```

POST Request

```bash
curl -X POST https://example.com
```

Download Request

```bash
curl -O https://example.com/robots.txt
```

## Arguments

Request Headers

```bash
curl --header 'Host: example.com' https://12.34.56.78
```

Include Status Code In Output

```bash
curl -i https://example.com
```
